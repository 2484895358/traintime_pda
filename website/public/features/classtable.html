Feature 01 / Classtable

课表页先服务今天，再服务整个学期。

课程地点、节次、周次与当天定位都是一眼先扫到的内容。布局强调阅读顺序，不靠海报式大字制造存在感。

地点优先周次切换午休 / 晚休分段

Reading Order

先看今天有没有课，再看地点和时间。

页面把最高频的问题排在前面，而不是把所有课程字段同权展示。

Layout

按真实上课节奏分段。

午休与晚休被明确标记，阅读节次时不需要自己在脑子里重新切段。

Mobile

地点文本优先保证能看清。

极端长度时先压缩老师信息，不先牺牲地点可读性。

In App

进入应用后，这个模块主要解决三件事。

  • 快速判断今天、下一节和本周课程安排。
  • 在手机上清楚看见地点，而不是被多余字段抢掉空间。
  • 保留周次切换与左右滑动，不再添加打断视线的额外状态入口。
